step1:卷积运算($\star$)¶

$$s(t)\star f(t)=\sum_{\tau=-\infty}^{\infty}x(\tau)h(t-\tau)$$¶

举个例子¶

s=[1,2,3], f=[1,2] 卷积过程

2D卷积运算(x尺寸为$N*M$)¶

$$x(n1,n2)\star h(n1,n2)=\sum_{k2=0}^{N-1}\sum_{k1=0}^{M-1}x(k1,k2)h(n_1-k_1,n_2-k_2)$$¶

$$0\leq n1 \leq N-1,0\leq n2 \leq M-1$$¶

## 定义一个对图像加高斯模糊的噪声程序
def add_noise(I):
    '''函数输入为灰度图像，输出为加过高斯白噪声的图像。'''
    mean = 0 ;var = 300
    sigma = var**0.5 
    row,col = I.shape[0],I.shape[1]
    gauss = np.random.normal(mean,sigma,(row,col)) 
    gauss = gauss.reshape(row,col) 
    I_noise = I + gauss 
    return I_noise
